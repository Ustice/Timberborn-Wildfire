# Changelog

All notable player-facing and release-facing Wildfire changes should be recorded here.

Wildfire uses Timberborn-style four-part versions in `manifest.json`. Release tags should use the same version with a leading `v`, for example `v0.1.0.0`.

## [0.1.0.0] - Unreleased

### Added

- Initial release package discipline for the Timberborn Wildfire mod.
- Release packaging validates the manifest version, package ZIP version, and changelog entry before reporting a package ready for handoff.
